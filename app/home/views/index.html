Simple to Use

Get started with just a few commands

# Start development servers
cd ui/ && npm run dev
go run main.go
# Add shadcn/ui components
npx shadcn-ui@latest add button
npx shadcn-ui@latest add dropdown-menu
# Build for production
npm run build
